Developer Friendly

Developed using latest technologies

Yet everything is so deeply and thoroughly documented.

React & MobX

Declarative, efficient, functional and scalable.

TypeScript

Strongly typed for better tooling and better developer experience.

CouchDB & PouchDB

A database system designed for synchronization, reliability and real-time application that scales from big data to mobile.

No Backend

It does not have or need any backend code, just connect it to a database server and it will work, so it can only have DBaaS as a backend, like IBM Cloudant.

Sass

Sass is the most mature, stable, and powerful professional grade CSS extension language.

Other...

Other libraries are used, all are reliable and thoroughly tested.

Offline / Multiple devices

The application can run perfectly offline, no internet connection is needed.

When an internet connection is present it will automatically upload the data to your server and the data will be in synchronization on all of the devices that are connected to the same server.

* Following the provided documentation, you'll be responsible for your server, which is basically just a CouchDB database.
